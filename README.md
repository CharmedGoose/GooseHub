# GooseHub (WIP)

A hub with geese.

## Running Locally

Clone this repository:  
`git clone https://github.com/CharmedGoose/GooseHub.git`

Then run:  
`deno install`

Fill in .env  

Rename .env.example to .env  

Then run:  
`deno task start`

Then open:  
`http://localhost:8000/`
