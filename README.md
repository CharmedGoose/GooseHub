# GooseHub (WIP)

A hub with gooses.

## Running Locally

1. Clone this repo:

    ```bash
    git clone https://github.com/CharmedGoose/GooseHub.git
    cd ./GooseGub
    ```

1. Fill in `.env.example`

Rename `.env.example` to `.env`

Then run:
`deno task start`

Then open:\
`http://localhost:8000/`
